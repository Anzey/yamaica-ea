Importing EA Projects

This 5 minutes tutorial will show you how to import Enterprise Architect models by transforming them to Franca IDL files.

Prerequisites

Before you start ensure that the following prerequisites are fulfilled:

  • You have installed a 32 Bit Java Virual Machine
  • You have the 32 Bit version of yamaica
  • You have read the UML Interface Standardization Guideline

Importing an EA Project

Click on the Import... button in the yamaica editor to open the import wizard. Select the Franca Interface Definition Files From Enterprise Architect Project entry from the list and press the Next button. Choose an Enterprise Architect Project file (e.g. this demo file) and select the packages you want to import. Next press the Finish button to create the Franca IDL file(s).
